# 📈 Dashboard DISC

Sistema web desenvolvido para análise comportamental de colaboradores internos e candidatos em processos seletivos, utilizando a metodologia DISC.

A aplicação coleta e organiza os resultados do questionário DISC, transformando as respostas em gráficos e indicadores visuais para facilitar a análise de perfil comportamental.

Esse dashboard auxilia gestores e recrutadores na tomada de decisão e no melhor direcionamento de pessoas dentro da empresa.

---

## 🧠 Sobre a Metodologia DISC

A metodologia DISC classifica perfis comportamentais em quatro fatores principais:

- **D - Dominância** → foco em resultados e desafios  
- **I - Influência** → comunicação e persuasão  
- **S - Estabilidade** → paciência e cooperação  
- **C - Conformidade** → organização e precisão
